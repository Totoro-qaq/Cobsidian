# Note Types

Use the smallest note shape that preserves the knowledge.

## Learning Note

Use for concepts, tools, papers, videos, or conversations that teach a reusable topic.

Recommended sections:

- Summary
- Core Concepts
- Workflow or Mental Model
- Common Mistakes
- Related Notes

## Project Note

Use for a specific project, experiment, incident, or implementation.

Recommended sections:

- Context
- Goal
- Implementation or Process
- Evidence
- Result
- Risks and Next Steps
- Related Notes

## Comparison Note

Use when the value comes from comparing options, versions, tools, or attempts.

Recommended sections:

- Short Conclusion
- Comparison Table
- Key Differences
- Decision Rules
- Related Notes

## Index Note

Use when multiple notes need a navigable entry point.

Recommended sections:

- Map
- Core Notes
- Project Notes
- Open Questions
